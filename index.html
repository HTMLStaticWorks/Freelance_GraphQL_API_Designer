Product Catalog with 500+ Queries/sec

The product listing page was triggering 1 query per product to fetch reviews. With 50 products per page, that's 51 DB calls per user request.

Before (N+1)
SELECT * FROM reviews WHERE product_id = 1; SELECT * FROM reviews WHERE product_id = 2; ... × 48 more queries
After (DataLoader)
